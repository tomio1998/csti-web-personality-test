## Overview

CSTI Game Personality Test is a browser-first quiz built around CS2 and CSGO player identity. It runs through 35 questions to shape a game personality profile, infer rank tendencies, and assign one of 15 personality types.

The project is meant for lightweight web use, with responsive layouts and multilingual content that help it adapt across devices and audiences. It blends a straightforward quiz flow with visual feedback plus audio and animation effects, so the results feel more interactive than a plain form.
